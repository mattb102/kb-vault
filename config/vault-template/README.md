# Vault template

This is the starter vault for a new vault. The setup process copies it
to your own vault folder and pushes it to a private GitHub repo, then your
Claude personalizes it.

The folder layout and frontmatter `type:` fields are load-bearing — the MCP
tools route by them. Safe to do: fill in the placeholder notes, add your own,
add more `metric-log` files, add `interest-overview` notes. Keep the
`AI-Observations/` files where they are (the AI tools find them by frontmatter).

See `config/CLAUDE.md.template` for the full routing table.
